# AMCP Governance Model / 治理模型

AMCP is a community-governed open protocol. No single organization controls it.  
AMCP 是一个社区治理的开放协议，没有任何单一组织控制它。

---

## Roles / 角色

| Role / 角色 | Description / 说明 | How to get / 如何获得 |
|------------|-------------------|----------------------|
| **User / 用户** | Uses AMCP implementations, files issues | Automatic / 自然参与 |
| **Contributor / 贡献者** | Submits PRs, docs, translations / 提交PR、文档、翻译 | First merged PR / 首个合并PR |
| **Maintainer / 维护者** | Reviews and merges for specific modules / 特定模块审查合并 | Nomination by Core Team / 核心团队提名 |
| **Core Team / 核心团队** | Overall direction, governance decisions / 整体方向、治理决策 | Election / 选举 |
| **TSC** | Technical disputes, roadmap, RFC votes / 技术争议、路线图、RFC投票 | Elected from Core Team / 从核心团队选举 |
| **Ambassador / 大使** | Regional outreach, meetups / 区域推广、线下活动 | Application + contribution review / 申请+贡献审核 |

---

## Decision Making / 决策流程

### Day-to-day / 日常决策
Maintainers merge PRs for their modules without formal vote.  
维护者可无需正式投票合并其模块的 PR。

### Significant changes / 重大变更
Any change affecting the normative specification requires an RFC:  
任何影响规范性文档的变更都需要 RFC 流程：

```
1. Open Discussion with [RFC Idea] tag / 开标记 [RFC Idea] 的 Discussion
2. Community feedback period: ≥14 days / 社区反馈期：≥14天
3. Submit formal RFC document to rfcs/ / 提交正式 RFC 文档到 rfcs/
4. TSC vote (simple majority) / TSC 投票（简单多数）
5. Implementation tracked in issues / 实现进度通过 issue 追踪
```

### Conflict resolution / 冲突解决
TSC has final authority on technical disputes. TSC decisions are recorded publicly.  
TSC 对技术争议拥有最终权威。TSC 决策均公开记录。

---

## Anti-Monopoly Rules / 反垄断规则

- No single organization may hold more than **1/3 of TSC seats** / 单一组织不得持有超过 **1/3 的 TSC 席位**
- The protocol specification may never be made proprietary / 协议规范永远不得变为专有
- The protocol specification is licensed **Apache 2.0 — permanently and irrevocably** / 协议规范采用 **Apache 2.0 — 永久且不可撤销**

---

## Meetings / 会议

- **Monthly community sync / 月度社区同步**: First Thursday UTC 14:00
- **TSC meeting / TSC会议**: Quarterly / 每季度一次
- All meeting notes published in [Discussions](https://github.com/stepmind/amcp/discussions) / 所有会议纪要发布在 Discussions
