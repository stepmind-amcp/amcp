# AMCP Technical Roadmap / 技术路线图
## 2026 — 2028

---

## Phase 1 — Foundation / 阶段一：协议奠基
**2026 Q2–Q3**

目标：让第一批实现者能够接入并通过 Level-1 一致性测试。  
Goal: Enable first implementers to connect and pass Level-1 conformance tests.

- [x] AMCP v1.0.0 specification published / v1.0.0 规范发布
- [x] Machine-readable spec (JSON) / 机器可读规范（JSON）
- [x] Community website live / 社区网站上线
- [ ] `amcp-python` Level-1 reference implementation / Python Level-1 参考实现
- [ ] Level-1 conformance test runner / Level-1 一致性测试运行器
- [ ] AMCP Registry launched / AMCP 注册表启动
- [ ] First external Level-1 passing implementation / 第一个外部通过 Level-1 的实现
- [ ] `docs/quickstart.md` with runnable examples / 含可运行示例的快速开始文档

---

## Phase 2 — Ecosystem / 阶段二：生态扩展
**2026 Q4 — 2027 Q2**

目标：多语言 SDK、主流存储适配器、AMQL 优化器。  
Goal: Multi-language SDKs, major storage adapters, AMQL optimizer.

- [ ] `amcp-go` Level-1 implementation / Go Level-1 实现
- [ ] `amcp-ts` Level-1 implementation / TypeScript Level-1 实现
- [ ] Level-2 conformance tests defined and runnable / Level-2 一致性测试定义并可运行
- [ ] Storage adapters: SQLite / Postgres / Pinecone / Weaviate / Chroma
- [ ] AMQL optimizer v1 / AMQL 优化器 v1
- [ ] Community plugin hackathon / 社区插件大赛

---

## Phase 3 — Enterprise / 阶段三：企业级增强
**2027 Q3 — 2028 Q1**

目标：安全、合规、联邦部署、性能基准。  
Goal: Security, compliance, federated deployment, performance benchmarks.

- [ ] ABAC fine-grained access control spec / ABAC 细粒度访问控制规范
- [ ] Cross-org federated memory sharing / 跨组织联邦记忆共享
- [ ] Performance benchmark suite / 性能基准测试套件
- [ ] Level-3 conformance tests defined / Level-3 一致性测试定义
- [ ] AMCP Certification Program launch / AMCP 认证计划启动
- [ ] AMCP v1.1 release (backward-compatible) / AMCP v1.1 发布（向后兼容）

---

## Phase 4 — Standard / 阶段四：行业标准
**2028+**

目标：行业标准，无限进化。  
Goal: Industry standard, infinite evolution.

- [x] License: Apache 2.0 from day one / 许可证：从第一天起即为 Apache 2.0 ✓
- [ ] AMCP Standards Alliance formation / AMCP 标准联盟成立
- [ ] ISO/IEC standardization submission / ISO/IEC 标准化提交
- [ ] AMCP v2.0 with neuro-symbolic memory / AMCP v2.0 含神经符号记忆
- [ ] Differentiable retrieval research integration / 可微分检索研究集成
